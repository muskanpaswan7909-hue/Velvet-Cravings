# Velvet Cravings 🍰

## 📌 Introduction
Velvet Cravings is an online bakery website where users can browse pastries, place orders, and manage their accounts easily.

## 🎯 Objectives
- Provide easy online food ordering
- Manage customer data efficiently
- Maintain product inventory

## 💻 Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## 🔐 Modules
1. User Authentication (Login/Signup)
2. Product Browsing
3. Cart System
4. Order Management
5. Admin Panel

## 📊 Data Flow Diagram (DFD)
User → Website → Database  
Admin → Manage Products → Database  

## 🧩 ER Diagram
Entities:
- User
- Orders
- Products
- Admin

Relations:
- User places Orders  
- Orders include Products  

## 🔥 Features
- Responsive design 📱
- Secure login system 🔐
- Online order placement 🛒
- Admin dashboard ⚙️
